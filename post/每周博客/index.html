1、ArrayList中按数组下标添加元素

public boolean add（int index ， T  t）{
	//throw exception if index is illegal
    if(index < 0 || index >= arr.length){
        throw new IllegalArgumentException("index is Illegal");
    }
    //expand when array is full
    if(size = arr.length){
        int newLen = getLen();
        grow(newLen);
    }
    //move array elements
    for(int i = size;i > index;i--){
        //从前向后遍历,否则出错
        //这里arr[size]经过扩容后一定存在
        arr[i] = arr[i - 1];
    }
    arr[index] = t;
    size++;
    modCount++;
    return true;
}

2、将String字符串转化为char数组并传入ArrayList集合中,最后返回该集合

private static ArrayList<Character> stringtoList(String word, ArrayList<Character> list){
        char[] o = word.toCharArray();
        Character[] p = new Character[word.length()];
        for (int i = 0; i < o.length; i++) {
            p[i] = o[i];
        }
        list = new ArrayList<>(Arrays.asList(p));
        return list;
    }

3、谨慎使用nextInt()与nextLine()

public class test01 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入第一个整数:");
        int i = sc.nextInt();
        System.out.println("请输入第一个字符串:");
        String s1 = sc.nextLine();
        System.out.println("请输入第二个字符串:");
        String s2 = sc.nextLine();
    }
}
//输出结果(从键盘输入的值为1,"1"):
请输入第一个整数:
1
请输入第一个字符串:
请输入第二个字符串:
"1"

关于该问题的解释:

API：nextInt():将输入的下一个标记扫描为int,读取完成之后将光标放在读取数据的同一行,改数据的后面

    nextLine():读取整行的数据包括单词间的空格,到回车结束,读取结束后,光标放在下一行开头。

nextInt只读取了数值1,剩下的“\n”还没有读取,并将光标放在本行中2的后面。接着nextLine会读取“\n''，并结束

本次读取。

解决方法:

​ 1、使用nextLine()“吃掉”nextInt()剩下的空格,如上段代码所示

​ 2、不使用nextInt(),要读取整型数据时,使用nextLine读取,用parseInt方法进行转换(推荐)
